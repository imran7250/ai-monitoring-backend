# AI Monitoring SaaS Backend

Production-ready backend built with Spring Boot for real-time monitoring, anomaly detection, and incident management.

## Features
- Health check monitoring with anti-flapping logic
- AI-based anomaly detection (3σ threshold)
- Incident lifecycle (trigger, acknowledge, resolve)
- Alert rules with notifications
- JWT authentication + RBAC
- Scheduled jobs & data retention

## Tech Stack
- Java, Spring Boot
- Spring Security, JPA/Hibernate
- MySQL (Railway)
- Deployed on Render

## Deployment
- Backend: Render
- Database: Railway
- Frontend: Vercel
